# Scraper.

## Acessar o ambiente

Para acessar o ambiente de desenvolvimento, abra o terminal e digite:

```bash
cd crawler-redacao-online-tema
source bin/activate
```

## Sair do ambiente

Para sair do ambiente de desenvolvimento digite:

```bash
deactivate
```

## Usage
